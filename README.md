# Social Wallet
A mobile app to display your social links like a digital wallet and share them with others with a single tap.

## Features
- Add and manage your social links in one place.
- Share your social wallet with others using a unique QR code or link.
- Customize your wallet with different themes and layouts.
- View and interact with your social links directly from the app.

## Installation
1. Clone the repository:
   ```bash
   git clone
    ```
2. Navigate to the project directory:
    ```bash
    cd social-wallet
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Run the app:
    ```bash
    npm start
    ```
